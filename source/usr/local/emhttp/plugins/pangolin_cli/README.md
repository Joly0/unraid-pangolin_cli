**Pangolin CLI**

Installs the [Pangolin CLI](https://docs.pangolin.net/manage/clients/install-client#pangolin-cli-linux-macos-windows) (`pangolin`) and adds a settings page to connect this Unraid server to your Pangolin instance as a **machine client**, so private resources can be reached on this host.

Configure it under **Settings > Pangolin CLI**:
- Enter your Pangolin endpoint URL, client ID and client secret (create a machine client in the Pangolin dashboard).
- Click **Apply** to connect, or enable **Start automatically on boot**.

The CLI is also available from the Unraid terminal as `pangolin`.

Source: https://github.com/fosrl/cli
